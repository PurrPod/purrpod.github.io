# 社区生态

PurrCat 致力于打造一个开放、共享的本地化 Agent 生态。

## 1. Skill 与插件集市 (建设中)

官方生态库正在筹备中，未来你可以在这里一键浏览或拉取由社区开发者提供的：
- 多种多样的 Skill
- 更加丰富有趣的 Sensor
- 各种 MCP 工具
- 针对特定行业工作流定制的 Harness 工作流与拓展节点

## 2. 拓展交互玩法

- 桌宠插件
- 零代码拖拽开发与部署工作流
- 可视化记忆图谱

## 3. 交流与反馈

- **Bug 追踪 & 功能建议**：请移步至 GitHub Issues 专区。提交前请善用搜索功能，避免重复提问。
- **经验分享**：欢迎在 GitHub Discussions 区分享你的调优心得或有趣的实验。

## 4. 共建指南

只要符合 GPL-3.0 协议，我们非常欢迎开发者 Fork 本仓库，无论是修复底层问题，还是提交一个新的拓展插件，社区都会及时进行 Code Review 并合并优质代码。
