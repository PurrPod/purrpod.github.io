# 社区生态

CatInCup 致力于打造一个开放、共享的本地化 Agent 生态。

## 1. Skill 与插件集市 (建设中)

官方生态库正在筹备中，未来你可以在这里一键拉取由社区开发者提供的：
- 效率办公 Skill 模板
- 数据分析与生信工具 Plugin
- 特定学术网站的 Sensor

## 2. 交流与反馈

- **Bug 追踪 & 功能建议**：请移步至 GitHub Issues 专区。提交前请善用搜索功能，避免重复提问。
- **经验分享**：欢迎在 GitHub Discussions 区分享你的 Harness Engineering 调优心得或有趣的沙盒实验。

## 3. 共建指南

只要符合 GPL-3.0 协议，我们非常欢迎开发者 Fork 本仓库，无论是修复底层的 Docker 挂载问题，还是提交一个新的飞书 Sensor，社区都会及时进行 Code Review 并合并优质代码。
