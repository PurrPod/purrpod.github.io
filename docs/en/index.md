## Quick Start

CatInCup is a local-first intelligent agent framework designed to provide secure, efficient, and extensible AI assistant capabilities. Through this documentation, you can learn how to use and extend CatInCup.

### Core Features

- **Local-First**: Data is completely localized, with secure local file operation capabilities
- **Minimal & Efficient**: Eliminates redundant prompts, router-based tool calls do not disrupt context order
- **Secure Isolation**: Pure Docker sandbox execution environment with whitelist permissions
- **Highly Customizable**: Minimal Python build, supports no-code extensions
